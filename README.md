# TP – Convertisseur Température & Distance (Fragments + Onglets)

Ce TP consiste à développer une application Android utilisant des Fragments associés à un système d’onglets (Tabs). L’objectif est de créer une interface simple permettant d’effectuer deux types de conversions :

Conversion de températures : Celsius (°C) ↔ Fahrenheit (°F)

Conversion de distances : Kilomètres (Km) ↔ Miles

L’application doit intégrer : un menu “Quitter” permettant de fermer l’application proprement ;

la gestion de la touche Retour afin d’afficher un dialogue de confirmation avant la fermeture.
